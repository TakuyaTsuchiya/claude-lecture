# Claude Code入門・実践 — 完了条件

## 課題

Claude Codeを使って、自分の業務・趣味・日常のいずれかで「1件の成果物または自動化」を作って提出してください。

## 提出内容

1. 何を作ったか（1〜2行）
2. 使ったプロンプトの抜粋
3. 成果物のスクリーンショット or URL（GitHub / Vercel等）
4. 詰まった点と解決方法（1〜3行）

## 運用方針

- **自動受理**: 提出が確認できた時点で講座完了扱い
- **事前の合否判定はなし**: 受講生の心理的ハードルを最小化
- **事後の軽いチェック**: 後日、事例として紹介したい場合のみ個別連絡

## 設計意図

- 受講生が実際に手を動かす機会を担保する
- 提出物を事例集として再利用する（営業資料・次期講座の改善・受講生紹介）
- 応用講座の受講上限管理と連動させる
