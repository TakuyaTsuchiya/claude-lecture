## [P1] カバー — MCP②Notion連携　組織の記憶を横断して経営課題を抽出する

今回はNotionと連携し、議事録データベースとプロジェクト管理データベースを横断検索します。

題材は、架空の航空会社「ドレミファ・エアー」の経営会議用の資料作りです。議事録とプロジェクト管理の2つのデータベースを横断して情報を収集し、HTMLダッシュボードにまとめます。

経営者・組織開発・人事など、複数の会議や部門の情報を統合して判断する立場の方にとってとても便利なツールです。

---

## [P2] Notionに蓄積された情報をMCP連携で自動的に取得しにいく

MCPを使うと、ClaudeがNotionに直接アクセスして議事録やプロジェクト記録を読みに行きます。

今日は、Notion公式のMCPを使ってこの横断検索を実演します。

---

## [P3] 2コマンドでインストール、新しいセッションで即繋がる

接続のセットアップはこのスライドの通りです。2コマンドでインストールして、新しいセッションを開き直すだけで繋がります。

注意点が一つあります。これらのコマンドはデスクトップ版のチャット欄では動きません。ターミナルを開いて `claude` と打ち込み、Claude Code CLIを起動した中で実行してください。

まず `/plugin marketplace add makenotion/claude-code-notion-plugin` でNotionのプラグインマーケットプレイスを追加します。続けて `/plugin install notion-workspace-plugin@notion-plugin-marketplace` でインストールします。

インストール後は必ず新しいセッションを開き直してください。ClaudeはセッションをロードするタイミングでMCPのツール定義を読み込むので、同じセッションのままでは反映されません。

新しいセッションを開いたら、`/mcp` を実行してNotionを認証します。「Authenticate」を選ぶとブラウザが開くので、NotionワークスペースへのアクセスをOAuthで許可します。これで接続完了です。
