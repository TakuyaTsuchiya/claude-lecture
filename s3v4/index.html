MCP②Notion連携
組織の記憶を横断して
経営課題を抽出する

WHAT MCP DOES

Notionに蓄積された情報を
MCP連携で自動的に取得しにいく

議事録もプロジェクト記録も、ClaudeがNotionを直接読みに行く。
SETUP

2コマンドでインストール、
新しいセッションで即繋がる

① /plugin marketplace add makenotion/claude-code-notion-plugin
→ マーケットプレイスを追加
② /plugin install notion-workspace-plugin@notion-plugin-marketplace
→ インストール実行
③ 新しいセッションで /mcp → Notion を認証
→ ブラウザでNotionへのアクセスを許可
必ず新しいセッションを開き直す
THE TASK

2つのデータベースを
一度に巡回する

議事録データベース
経営会議 × 3回
事業部会議 × 4回
顧客面談 × 3回
プロジェクト管理データベース
LCC対抗施策
機材更新計画
新路線就航・その他
PROMPT
来週の経営会議で使う資料を作ってください。Notionの「議事録」データベースから過去3ヶ月の議事録を読み込み、経営課題を洗い出してください。次に「プロジェクト管理」データベースから各プロジェクトの進捗を確認し、課題とプロジェクトを紐づけてHTMLダッシュボードにまとめてください。

数時間の作業が、
数分で終わる

Notion連携で、組織に積み上がった記憶を
意思決定に直結させる。
