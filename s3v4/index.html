THE PROBLEM

情報は蓄積されている。
でも、引き出せない。

SLACK
流れる情報
リアルタイム把握
NOTION
蓄積された記憶
組織全体の横断検索
SETUP

2コマンドでインストール、
ワンクリックで認証

① /plugin marketplace add
→ 一覧からNotionを選択
② /plugin install
→ 再起動 または /reload-plugins で有効化
③ /mcp
→ ブラウザでワークスペースを選んで「アクセスを許可」
THE TASK

2つのデータベースを
一度に巡回する

議事録データベース
経営会議 × 3回
事業部会議 × 4回
顧客面談 × 3回
プロジェクト管理データベース
LCC対抗施策
機材更新計画
新路線就航・その他
PROMPT
過去3ヶ月の議事録を横断して、経営課題に該当するトピックを抽出してください。それぞれの課題が、プロジェクト管理データベースのどのプロジェクトと関連するかも紐付けてください。最後に、課題別の議事録リンク・優先度・繰り返しパターンをまとめたHTMLダッシュボードを生成してください。
