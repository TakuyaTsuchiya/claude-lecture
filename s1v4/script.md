今日のお題は、社内の会議資料作りです。

---

## [P2] 今日のお題 — アジェンダのMDファイルを渡すと会議用HTMLサイトが完成する

今日やることを一言で言うと、「アジェンダのMarkdownファイルを Claude Code に渡して、会議用のHTMLサイトに変換する」です。

実際の仕事でも、会議のアジェンダを事前にまとめることはよくありますよね。そのアジェンダを、Claude と打ち合わせしながら決めて、Markdownファイルにしておく。それを Claude Code に渡すだけで、ブラウザで開ける会議資料に仕上がります。

今日は、架空のディスカウントストア「驚安堂」の2026年夏・新商品リリース説明会のアジェンダを事前に用意しました。このファイルを使って実演します。

---

## [P3] プロンプト全文

投げるプロンプトはこれだけです。

> あなたは驚安堂の社員です。@agenda.md のアジェンダをもとに、会議用のHTMLページ1枚にまとめてください。

`@agenda.md` は、ファイルをそのまま渡す書き方です。Claude Code でフォルダを開いた状態でこう書くと、そのファイルの中身をまるごと読み込んでくれます。

役割を一言添えて、あとは「@ファイル名」で渡すだけ。それだけです。

では実際にやってみます。

## 【実演パート】Claude Code で会議資料を生成

（画面を Claude Code デスクトップ版に切り替え、agenda.md があるフォルダを開く）

まずフォルダを開きます。agenda.md が入っているフォルダを選択します。

（プロンプトをペーストして送信）

`@agenda.md` と打つと、候補としてファイル名が出てきます。選択して送信します。

（Claude Code が動き始める様子を見せる）

Claude Code がアジェンダを読み込んで、HTMLに組み上げていきます。

この待ち時間、皆さんは別の仕事をしていて構いません。

（HTML生成が完成するまで待機）

できあがりました。生成された HTML をブラウザで開きます。

（ブラウザで開いて、ページを上から下までスクロールして見せる）

表紙、アジェンダ、商品10点、販売チャネル、KPI、スケジュール、アクションアイテム……アジェンダの構造がそのままきれいなHTMLになっています。商品カードには「ジェットコーラ 88円」「光るぞ☆オシキン」と、アジェンダに書いた内容が並んでいますね。

Markdownファイルを1つ渡しただけで、ここまで仕上がりました。

## [P4] クロージング — 設定より、まず手を動かそう

はじめて Claude Code を使ってみて、どうでしたか？

細かいことは、あとから覚えれば十分です。

Claude Code の使い方を覚えるのに一番いい方法は、使いながら覚えることです。分からないことが出てきたら、そのまま Claude Code に聞けばいい。詰まったら「こういうエラーが出ました」と貼り付けて聞けばいい。

設定や機能を全部理解してから動こうとすると、そこで止まってしまいます。

まず手を動かして、詰まったらその都度調べる。これが、Claude Code を一番速く使えるようになる道です。
