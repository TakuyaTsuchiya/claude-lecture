## [P1] カバー — 社内会議用資料をHTMLでサクッと作ってみよう

前回、インストールは完了しましたか？

今回からいよいよ、皆さん自身の手で Claude Code を動かしていきます。

V2では、私がプロンプトを投げて、Claude Code が動く様子をご覧いただきました。今回は同じことを、皆さんにもやってもらいます。

今日のお題は、社内の会議資料作りです。

---

## [P2] 今日のお題 — あなたは驚安堂の社員。会議用の資料を作ってください。

皆さんに、架空のディスカウントストア「驚安堂」の社員になってもらいます。

来月、驚安堂は夏の新商品リリース説明会を開きます。PBブランドの新商品10品目を一斉に投入する、大事な社内説明会です。

その会議資料を、Claude Code に作ってもらいます。URLを1本送るだけで、参加者全員がブラウザで開ける、そういう会議資料を目指します。

---

## [P3] プロンプト全文

投げるプロンプトはこれだけです。

> あなたは驚安堂の社員です。来月、2026年夏の新商品リリース説明会を開きます。PBブランド「マジ価格NEO」10商品・販路・SNS施策・KPI・スケジュールを含む会議用の資料を、URL1本で参加者に配布できるHTMLページ1枚にまとめてください。

「あなたは〇〇の社員です」と役割を与えて、やってほしいことを日本語で書く。指示はこれだけです。

会議で何を話すのか、どんな商品ラインアップにするのか、KPIはどう置くのか。細かい中身は Claude Code が自分で考えてくれます。

では実際にやってみます。

## 【実演パート】Claude Code で会議資料を生成

（画面を Claude Code デスクトップ版に切り替え）

Claude Code を起動して、プロンプトをペースト、送信します。

（送信して、Claude Code が動き始める様子を見せる）

Claude Code は、まず会議のアジェンダから組み立てます。開会の挨拶、市場背景、商品ラインアップ、販売チャネル、プロモ施策、KPI、スケジュール、各部へのお願い……説明会に必要な項目を自分で考えて、驚安堂らしい商品名や価格まで作り込んでいきます。

この待ち時間、皆さんは別の仕事をしていて構いません。

（HTML生成が完成するまで待機）

できあがりました。生成された HTML をブラウザで開きます。

（ブラウザで開いて、ページを上から下までスクロールして見せる）

表紙、アジェンダ、商品10点、販売チャネル、KPI、スケジュール、アクションアイテム……すべて1枚のページに収まっています。商品カードには「ジェットコーラ 88円」「光るぞ☆オシキン」と、驚安堂らしい名前まで並んでいますね。

社員という役割と、お題を一文で渡しただけで、ここまで仕上がりました。あとはこのURLを社内で共有すれば、参加者全員がブラウザで開けます。

## [P4] クロージング — 設定より、まず手を動かそう

はじめて Claude Code を使ってみて、どうでしたか？

「アジェンダの中身までClaudeが考えるの？」「もっと細かく指示しないといけないのでは？」そう感じた方もいるかもしれません。

細かいことは、あとから覚えれば十分です。

Claude Code の使い方を覚えるのに一番いい方法は、使いながら覚えることです。分からないことが出てきたら、そのまま Claude Code に聞けばいい。詰まったら「こういうエラーが出ました」と貼り付けて聞けばいい。

設定や機能を全部理解してから動こうとすると、そこで止まってしまいます。

まず手を動かして、詰まったらその都度調べる。これが、Claude Code を一番速く使えるようになる道です。
