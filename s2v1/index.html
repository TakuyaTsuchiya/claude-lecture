CLAUDE.mdの書き方

CLAUDE.md とは

AIが毎回読む「取扱説明書」

①
作業フォルダの一番上に置く CLAUDE.md ファイル
②
Claude Code がセッション開始時に自動で読み込む
③
「どう振る舞ってほしいか」を一度書けば毎回有効
書き始め

まず 3行 でOK

プロジェクト概要：〇〇を目的としたWebアプリです。
説明スタンス：ですます調で。技術用語には補足説明を入れてください。
参照ファイル：よく使う資料は「docs/company.md」に置いています。
①プロジェクト概要 ── 何を作っているかを伝える
②説明スタイル ── 文体・用語の扱いを指定する
③ファイルの場所 ── よく使う資料のパスを伝える
ショートカット

/init で自動作成

init = initialize（初期化する）の略
→ 「最初のセットアップをする」という意味
DEMO

実際に /init を使ってみる

①
Claude Code で対象のフォルダを開く
②
/init を実行 → CLAUDE.md が自動生成
③
3行の指示を追記して保存
活用法

ドキュメントのリンク集として使う

## よく使うドキュメント

- [会社概要](docs/company.md)
- [商品カタログ](docs/products.md)
- [メール文体ガイド](docs/email-tone.md)
必要なときに Claude Code が自動で参照しにいく

CLAUDE.mdは短く保つこと

長いほど AI は指示を守らなくなる
目安は 200行以内
増えたら .claude/rules/ に分割して呼び出す
