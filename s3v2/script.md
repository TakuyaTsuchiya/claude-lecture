# S3V2 台本 — 【経理・経営企画向け】月次レポート作成をSkill化しよう

- **想定尺**: 約8〜10分（うち実演 約7〜8分）
- **実演**: あり
- **成果物**: Skill Creatorで月次経営報告レポートSkillを作り、ドレミファエア社3月度のHTMLレポートを生成する

---

## [P1] カバー — 【経理・経営企画向け】月次レポート作成をSkill化しよう

今日は、自分の業務をSkill化して、資産にする方法を学びます。
経営報告レポート作成のような定型業務をSkill化、自動化してチームの資産にしましょう。

---

## [P2] 今日のお題

題材は架空の中堅航空会社、ドレミファエア社です。羽田を中心に7路線を運営しています。今月は春休み需要で売上が回復しましたが、LCC参入の懸念も浮上しています。

経理・経営企画の担当者が毎月やっている作業があります。財務システムから数字を取り出して、経営会議の議事録を読み返して、路線ごとの事業背景を整理して、経営者向けの月次レポートにまとめる。毎月同じ手順で、毎月10〜30時間かかります。しかも完成したレポートは数字の羅列になりがちで、経営会議では読まれない。

今日はこの作業を丸ごとSkillに覚えさせます。インプットは3つのファイルだけです。財務数値のCSV、経営会議の議事録、路線概要や競合動向の事業背景メモ——この3つを渡すだけで、背景と示唆を含む経営者向けHTMLレポートが出てくるようにします。

---

## [P3] Skill Creatorのしくみ

これを実現するのがSkill Creatorです。

自分の作業手順をClaudeに対話で説明すれば、Skill仕様に変換してくれます。「何をさせたいか」「いつ動かすか」「何を出力するか」——この3つを答えるだけです。コードは書きません。

ただし、1回の対話で完璧なSkillができるわけではありません。最初は叩き台を作り、実際に動かして、気になる箇所を対話で手直ししていきます。対話を重ねるほど、Skillは自分の業務にフィットしていきます。

---

## 【実演パート】月次レポートSkillを作る

（スライドを閉じてClaude Code CLIのターミナル画面に切り替え）

まず `demos/doremifa-air/` の3ファイルを確認します。

（`/document-skills:skill-creator` と入力して起動）

Skill Creatorが起動すると、すぐに「何を作りたいか」を聞いてきます。月次経営報告レポートを自動生成するSkillを作りたいこと、インプットは3ファイルで、アウトプットはHTMLレポート、「月次レポートを作って」というフレーズでトリガーする——これを対話で答えます。

（対話を進める）

Skill Creatorが設計の確認を続けます。チャートライブラリの種類、レポートに含める要素、前年比データがない場合の処理——こうした細かい仕様も対話で決まっていきます。

（`.claude/skills/monthly-business-report/SKILL.md` が生成されたことを確認）

SKILL.mdが生成されました。自分の頭の中にあった手順が、ファイルとして外部化されました。

ここで一度セッションを終了して、新しいセッションを開きます。Skillはセッション開始時に読み込まれるので、このひと手間が必要です。

（新しいセッションを開く）

「月次レポートを作って」と入力します。

（`demos/doremifa-air/` の3ファイルを自動で読み込む様子を見せる）

`demos/doremifa-air/` の3ファイルを自動で探して読み込みました。こちらからファイルを渡す必要はありません。

（`reports/2026-03.html` が生成されブラウザで確認）

売上51.3億円・前月比+20.7%、座席利用率78.1%・前年同月比+3.0ポイント——数字だけでなく、「LCC参入で羽田-札幌路線の客単価が下がるリスク」「羽田-松山新路線の就航前プロモーション」という経営への示唆も含まれています。

（気になる箇所があればClaudeに伝えて対話で微調整）

気になる箇所があれば、そのまま対話で直せます。「グラフの色を変えて」「示唆をもう一段踏み込んで」——こう伝えれば、Skill Creatorが SKILL.mdを書き換えてくれます。対話を重ねるほど、Skillは自分の業務にフィットしていきます。

このSkillは `.claude/skills/` に保存されています。プロジェクトのリポジトリに入れればチームの全員が同じように動かせます。ベテラン担当者が積み上げた手順が、組織全体で使えるSkillになります。

## [P4] クロージング — 手順の自動化 = 組織の資産

Skill Creatorを使えば、コードを書かなくても自分の業務手順をSkillに変換できます。

経理・経営企画に限らず、人事・営業・マーケティングの定型業務も同じです。毎月繰り返している作業を一つ思い浮かべてください。その手順を言葉にして説明するだけで、Skillになります。
