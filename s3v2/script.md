# S3V2 台本 — 【経営企画向け】月次レポート作成をSkill化しよう

- **想定尺**: 約8〜10分（うち実演 約7〜8分）
- **実演**: あり
- **成果物**: skill creatorで社長会議用の月次報告レポートをHTMLで出力するSkillを作る。

---

## [P1] カバー — 【経営企画向け】月次レポート作成をSkill化しよう

今日は、自分の業務をSkill化する方法を学びます。
定型業務である社長会議向けの月次レポート作成をSkill化してチームの資産にしましょう。

---

## [P2] 今日のお題

題材は架空のエアライン、ドレミファエア社です。羽田を中心に7路線を運営しています。

経営企画の担当者は財務システムから数字を取り出して、経営会議の議事録を読み返して、路線ごとの事業背景を整理して、社長会議向けの月次レポートにまとめています。毎月同じ手順で、3〜4営業日かけているので自動化して生産性を上げたいです。

今日はこの作業を丸ごとSkill化します。インプットは財務数値のCSV、経営会議の議事録、路線概要や競合動向の事業背景メモの3つで、これらをもとに示唆に富む社長会議向けの月次レポートをHTMLで生成します。

---

## [P3] Skill Creatorのしくみ

Skillを作るためにはAnthropic社公式のSkill CreatorというSkillを使います。
Skillを作るためのSkillなので少しややこしいですが、Claude Codeで対話しながらSkillを作ることができます。


---

## 【実演パート】月次レポートSkillを作る

（スライドを閉じてClaude Code デスクトップ版の新しいセッションに切り替え）

なお、Skill Creatorは事前にインストールが必要です。ターミナルでclaudeを起動し、こちらのコマンドを一度実行します。
（/plugin install skill-creator@claude-plugins-official）

（チャット欄に「新しいSkillを作りたい」と入力して送信）

/reload-pluginsでリロードします。
これでSkill Creatorがインストールされましたので、新しいセッションで「新しいSkillを作りたい」と伝えます。
するとSkill Creatorが自動で起動します。
Claudeが文脈を読んで、必要なSkillを自律的に判断して呼び出しているのです。

「どんなSkillを作りたいですか？」に対して「社長会議向けの月次レポートを自動生成するSkillを作りたい。インプットはディレクトリに入っている3つのファイルで、アウトプットはシングルHTML形式、TailwindCSSでスタイリングしたレポート、『社長会議向けの月次レポートを作って』というフレーズでトリガーする」これをプロンプトとして与えます。

「いくつか確認させてください」と対話が始まるのでこれに答えます。
インプットについて確認されましたので、３つのファイルを渡しました。
確認事項が全部確認できたのでSkillを書き始めてくれています。

（対話を進める）


Skill化の作業が完了しました。SKILL.mdが生成されました。定型業務の手順が、ファイルとして外部化されました。

SKILL.mdは普通のMarkdownファイルです。「どんな言葉でトリガーするか」「何を入力とするか」「どんな出力を返すか」が日本語で書かれています。人間が読んで編集できるのが特徴です。

ここで一度セッションを終了して、新しいセッションを開きます。Skillはセッション開始時に読み込まれるので、このひと手間が必要です。

（新しいセッションを開く）

「社長会議向けの月次レポートを作って」と入力します。


（`reports/2026-03.html` が生成されブラウザで確認）

売上51.3億円・前月比+20.7%、座席利用率78.1%・前年同月比+3.0ポイント——数字だけでなく、「LCC参入で羽田-札幌路線の客単価が下がるリスク」「羽田-松山新路線の就航前プロモーション」という経営への示唆も含まれています。

（気になる箇所があればClaudeに伝えて対話で微調整）

気になる箇所があれば、そのまま対話で直せます。「グラフの色を変えて」「示唆をもう一段踏み込んで」——こう伝えれば、Skill Creatorが SKILL.mdを書き換えてくれます。対話を重ねるほど、Skillは自分の業務にフィットしていきます。

このSkillをプロジェクトメンバーに共有すればメンバー全員が同じ資料を生成できるようになります。

---

## [P4] クロージング — 手順の自動化 = 組織の資産

Skill Creatorを使えば、コードを書かなくても自分の業務手順をSkillに変換できます。

経営企画に限らず、経理・人事・営業・マーケティングの定型業務も同じです。毎月繰り返している作業を一つ思い浮かべてください。その手順を言葉にして説明することで、チーム全員で使えるSkillになります。
