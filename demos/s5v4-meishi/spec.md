# 名刺管理アプリ — 仕様書

## 目的

紙の名刺を撮りためる代わりに、ブラウザだけで名刺情報を整理・検索できる1人用アプリ。
Sansanの機能を最小構成で再現する（OCRは対象外）。

## 利用シーン

- 展示会で交換した名刺の情報を手入力でストックする
- 過去に会った人を会社名や氏名で検索する
- 連絡先やメモを後から編集・追記する

## データ項目（名刺1枚）

| 項目 | 必須 | 備考 |
|---|---|---|
| id | ○ | アプリが自動採番 |
| 氏名 | ○ | 例: 山田 太郎 |
| 会社名 | ○ | 例: 株式会社サンプル |
| 役職 | | 例: 代表取締役 |
| メール | | 形式バリデーションは不要（最小構成） |
| 電話 | | |
| メモ | | 複数行可。交換日や話した内容を書く |
| 作成日時 | ○ | アプリが自動付与 |

## 画面構成（ツーカラム）

画面は**左右2ペイン**で構成する。左は常時一覧、右は選択・操作に応じて内容を切り替える。

### 左ペイン（常時表示）

- 上部: 検索ボックス（氏名・会社名・役職を横断検索）と「＋ 新規登録」ボタン
- 下部: 名刺カードの縦リスト（スクロール可）
- 各カード: 氏名（大）／会社名／役職／アクセントカラーの縦バー
- 選択中のカードはハイライトする
- カードをクリックで右ペインに詳細を表示

### 右ペイン（モード切替: 空 / 詳細 / フォーム）

**モード1: 空状態**
- 何も選んでいない初期状態
- 「左から名刺を選んでください」のガイドを中央に表示

**モード2: 詳細**
- 上部: 右側に「編集」「削除」ボタン
- 中央: 氏名（大見出し）・会社名・役職
- 下部: メール・電話・メモ・登録日を行ラベル形式で表示
- 削除時は確認ダイアログを出す

**モード3: 登録・編集フォーム**
- 上部: 「キャンセル」ボタン／タイトル（新規登録 or 編集）
- 中央: 各項目の入力フィールド（氏名・会社名・役職・メール・電話・メモ）
- 下部: 「保存」ボタン
- 保存後は詳細モードへ切り替わる

## 操作フロー（CRUD）

左ペインは常に表示されている前提で、右ペインの状態が遷移する。

| 操作 | 右ペインの遷移 |
|---|---|
| 作成 (Create) | 空 or 詳細 → ＋新規登録 → フォーム → 保存 → 詳細 |
| 一覧 (Read) | 起動時に左ペイン表示。検索ボックスで絞り込み |
| 詳細 (Read) | カードクリック → 詳細 |
| 更新 (Update) | 詳細 → 編集 → フォーム（既存値入り）→ 保存 → 詳細 |
| 削除 (Delete) | 詳細 → 削除 → 確認 → 空状態 |

## 永続化

- `localStorage` のキー `meishi-cards` に配列として保存
- 起動時に読み込み、変更時に都度書き込む

## 初期データ

空だと触感がゼロなので、初回起動時にサンプル3件を投入する（localStorage未設定のときのみ）。
実在しない人物名・会社名にする。

## 対象外（今回作らない）

- OCR（画像からの自動入力）
- クラウド保存・他端末共有（S6でSupabase連携するときに拡張）
- ログイン・マルチユーザー
- インポート／エクスポート
- 並び替え（デフォルトは作成日時の新しい順）

## デザイン方針

- ツーカラムレイアウト（左=一覧リスト、右=詳細・フォーム）
- 白基調、アクセントはClaudeオレンジ `#c15f3c`
- 余白多め、角丸 `rounded-lg`、影は控えめ
- モバイル対応は今回スコープ外（PC表示を前提とする）
