## [P1] カバー — Vercel入門 アカウント作成と初めてのデプロイ

前回、GitHubにリポジトリを作りましたがそれだけでは、まだWeb上で見ることはできません。Webサイトの形で世界に公開するには、もう1つ、Vercelが必要です。

---

## [P2] VercelでWebサイトを公開しよう

Vercelは、GitHubで管理しているコードを、Webサイトとして世界に公開してくれるサービスです。

今日やることは3つです。Vercelのアカウントを作り、VercelとGitHubを連携し、前回作ったリポジトリをデプロイします。デプロイとは、リポジトリを公開することです。

---

## [P3] GitHub と Vercel

GitHubとVercelを1回つないでしまえば、Claude Codeがプッシュでコードを更新するだけで、Vercelが自動で最新のサイトに更新してくれます。

---

## [P4] プロンプト全文

まずは、前回作ったリポジトリに、シンプルなHTMLサイトを追加します。プロンプトはこちらです。長いので概要欄にも貼っておきます。

> my-first-repo に index.html を追加してください。「Claude Codeと作るはじめてのWebページです」という見出しと、短い挨拶文を入れた1画面のページにしてください。完了したらコミットとプッシュしてください。

## 【実演パート】index.html追加 → Vercelデプロイ → 公開URL確認

（Claude Code デスクトップ版に切り替え、プロンプトを送信。index.html作成〜コミット〜プッシュが完了）

完成しました。

（ブラウザで https://vercel.com を開く）

次にVercelです。公式サイトにアクセスして、Hobbyをクリック。名前を入力します。GitHubアカウントでそのままサインアップできるので、Continue with GitHubを選びます。権限の確認画面が出たらAuthorizeを押してください。

電話番号を入力してContinueを押してください。番号にショートメッセージが送られてきますので、その番号を入れて承認します。

ダッシュボードに入りましたら、Import Git Repository の中の Continue with GitHub を押します。メールに番号が届きますので、それをコピペして承認します。

my-first-repo を選んでImportをクリックしてください。

Deployをクリックします。ビルドが終わり、公開URLが発行されます。

こちらのURLをクリックすると、先ほどClaude Codeに作ってもらったページが、世界中からアクセスできる状態で表示されます。これで自分のWebページが世界に公開できました。

---

## [P5] クロージング — サイトが世界に公開されました

一度つないでしまえば、あとはClaude Codeにプッシュしてと頼むだけで、最新のサイトが自動で反映されます。
