## [P1] カバー — Claude Code を自分のPCで動かそう

前回は、Claude Code を使って回転寿司3社の比較資料を作る様子をご覧いただきました。

今回は、あの環境を皆さん自身のPCで再現します。

途中でわからないことが出てきたら、Claude や ChatGPT に「こういうエラーが出ているので解決方法を教えて」と状況を伝えれば、一緒に解決策を探してくれます。AIに相談しながらAIツールをインストールする、これ自体がすでにClaude Code の使い方の練習になります。

## [P2] 事前準備 — 必要なもの

この講座ではClaude Codeデスクトップ版を使用します。Anthropic公式のインストール手順ページからアプリをダウンロードしてください。リンクは概要欄に載せています。

Claude Code は**Proプラン以上**で使えます。仕事で日常的に使うなら**Maxプランがおすすめです**。Proプランに比べて利用上限が大きいので、長い作業でも途中で止まりにくくなります。

## 【実演パート】macOS インストール

（画面をMacのインストール操作に切り替え）

ダウンロードした `Claude.dmg` を開いて、アプリケーションフォルダにドラッグ。起動してサインインしたら、上部の **Code タブ**をクリックします。未課金の場合はここでアップグレードを促す画面が出るので、Proプラン以上を選択してください。

---

## [P4] Windows インストール

（画面をWindowsに切り替え）

続いてWindowsです。
Windowsの場合、Git for Windowsを先にインストールします。
公式ページのClick here to downloadをクリックして最新版をインストールします。

インストールできたらPowerShellを起動して `git --version` コマンドを実行します。
git version 2.54.0.windows.1など、その時の最新版が表示されれば成功です。

次にClaude Codeデスクトップ版をダウンロードします。
`Claude Setup.exe` を実行してインストール後、起動してClaudeにサインインします。
未課金の場合はMacと同様にサブスクリプションのアップグレードを促す画面が出るので、そこでProプランを選択してください。
