## [P1] カバー — Claude Code を自分のPCで動かそう

前回は、Claude Code を使って回転寿司3社の比較資料を作る様子をご覧いただきました。

今回は、あの環境を皆さん自身のPCで再現します。

インストールは5分もかかりません。この動画が終わるころには、皆さんのPCでもClaude Code が動いている状態になります。

途中でわからないことが出てきても、一人で抱え込まないでください。Claude や ChatGPT に「こういうエラーが出たんですけど」と状況を伝えれば、一緒に解決策を探してくれます。AIに相談しながらAIツールをインストールする、これ自体がすでにClaude Code の使い方の練習になります。

---

## [P2] 事前準備 — 必要なもの

まず確認していただきたいのは、プランです。

Claude Code は、**Proプラン以上**で使えます。仕事で日常的に使うなら、**Maxプランがおすすめです**。Proプランに比べて利用上限が大きいので、長い作業でも途中で止まりにくくなります。

プランの確認ができたら、公式のインストール手順ページを開いてください。スライドにURLを表示していますが、この動画の概要欄にもリンクを載せているので、そちらからすぐに開けます。

---

## [P3] macOS インストール

（画面を `https://code.claude.com/docs/ja/desktop-quickstart` に切り替え）

まずMacからです。ページ上のダウンロードリンクからMac版を取得します。

ダウンロードした `Claude.dmg` を開いて、アプリケーションフォルダにドラッグ。起動してサインイン、そして上部の **Code タブ**をクリックすれば完了です。

このページの右側に「アシスタント」パネルがあります。わからないことがあればここで直接質問できます。それ以外でも、Claude や ChatGPT に相談しながら進めてもらって大丈夫です。

## [P4] Windows インストール

（画面をWindowsに切り替え）

続いてWindowsです。同じページからWindows版をダウンロードします。

1点だけ注意があります。Windowsでローカルのファイルを扱う場合、**Git for Windows** が必要です。未導入の方は先にインストールしておいてください。`.exe` を実行してインストール後、起動してサインイン、Code タブをクリックで完了です。
