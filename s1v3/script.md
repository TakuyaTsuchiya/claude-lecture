## [P1] カバー — Claude Code を自分のPCで動かそう

前回は、Claude Code を使って回転寿司3社の比較資料を作る様子をご覧いただきました。

今回は、あの環境を皆さん自身のPCで再現します。

---

## [P2] 事前準備 — 必要なもの

まず、確認していただきたいことが1つあります。

Claude Code は、**Claude の Proプラン以上が必要**です。Freeプランでは使えません。

仕事で日常的に使うなら、**Maxプランを強くおすすめします**。Proプランはトークン上限が厳しく、長い作業途中でリセットされることがあります。Maxプランはその上限が大幅に緩いので、実務で使うなら最初からMaxにしておく方が余計なストレスがありません。

プランを確認できたら、`claude.ai/download` にアクセスしてください。

## [P3] Mac インストール

（画面を `claude.ai/download` に切り替え）

まずMacからです。ページを開くと、お使いのOSに合ったバージョンが自動で表示されます。

ダウンロードしたら `.dmg` ファイルを開いて、アプリケーションフォルダにドラッグするだけです。

## 【実演パート】Mac インストール

（画面をMacのインストール操作に切り替え）

ダウンロードから起動まで、実際にやってみます。

アプリを起動すると、サインイン画面が出ます。Claude.ai に登録済みのアカウントでサインインするだけです。

ターミナルの設定も、PATHの設定も、一切不要です。サインインしたらすぐ使えます。

## [P4] Windows インストール

（画面をWindowsに切り替え）

続いてWindowsです。手順はMacとまったく同じです。

`claude.ai/download` を開いて、Windows版をダウンロードします。`.exe` ファイルを実行してインストールし、起動してサインイン。以上です。

## 【実演パート】Windows インストール

（画面をWindowsのインストール操作に切り替え）

こちらも同様に進めます。

Node.js も Git for Windows も不要です。デスクトップ版はインストーラーがすべて内包しているので、追加の作業はありません。

サインインが完了すれば、前回お見せしたような操作がそのまま使えます。
