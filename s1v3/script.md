前回は、Claude Code を使って回転寿司3社の比較資料を作る様子をご覧いただきました。

---

## [P2] 事前準備 — 必要なもの

まず確認していただきたいのは、プランです。

Claude Code は、**Proプラン以上**で使えます。仕事で日常的に使うなら、**Maxプランもおすすめです**。Proプランに比べて利用上限が大きいので、長い作業でも途中で止まりにくくなります。

プランの確認ができたら、公式のインストール手順ページを開いてください。スライドに表示している `code.claude.com/docs/ja/desktop-quickstart` です。このページを見ながら進めます。

---

## [P3] macOS インストール

（画面を `code.claude.com/docs/ja/desktop-quickstart` に切り替え）

まずMacからです。ページ上のダウンロードリンクからMac版を取得します。

ダウンロードした `.dmg` を開いて、アプリケーションフォルダにドラッグ。起動してサインイン、そして上部の **Code タブ**をクリックすれば完了です。

## 【実演パート】macOS インストール

（画面をMacのインストール操作に切り替え）

実際にやってみます。

アプリを起動してサインインしたら、上部の「Code」タブをクリックします。ターミナルの設定も、PATHの設定も、一切不要です。

---

## [P4] Windows インストール

（画面をWindowsに切り替え）

続いてWindowsです。同じページからWindows版をダウンロードします。

1点だけ注意があります。Windowsでローカルのファイルを扱う場合、**Git for Windows** が必要です。未導入の方は先にインストールしておいてください。`.exe` を実行してインストール後、起動してサインイン、Code タブをクリックで完了です。

---

## 【実演パート】Windows インストール

（画面をWindowsのインストール操作に切り替え）

Windowsも同様に進めます。

Node.js の別途インストールは不要です。デスクトップ版のインストーラーが内包しているので、Git 以外に追加作業はありません。
