PROMPT

Claude Codeへ渡すプロンプト

# 入力
- timecard.xlsx（タイムカード 10名×3月分）
- applications.xlsx（申請 有給5・残業3・休出2）

# タスク
申請を ✅承認可 / ⚠️要確認 に仕分けしたHTMLレポート生成

# 判定ルール
- 有給：対象日に打刻がなければ承認可
- 残業：申請時間と実打刻の差15分以内で承認可
- 休出：申請時刻と実打刻の差15分以内で承認可
- 申請漏れ：非営業日の申請なし打刻を別枠で検出

# 出力
output/check-report.html
CLOSING
