# S5V4 台本 — 名刺管理アプリを自作しよう

- **想定尺**: 約5分（うち実演 約3.5分）
- **実演**: あり

---

## [P1] カバー — 名刺管理アプリを自作しよう

この動画では、名刺管理アプリを1本、自前で作ります。使うのはアプリ開発用のCLAUDE.mdと、仕様書であるspec.md。Claude Codeにこの2つを渡して、1時間以内に動くアプリを仕上げるのがゴールです。

---

## [P2] TODAY'S GOAL — スペック駆動でアプリ開発を体験する

流れは3ステップです。アプリ開発用のCLAUDE.mdを書き、spec.mdで画面と機能を固め、Claude Codeに実装を頼みます。

アプリ開発にはスペック駆動という考え方があります。先に仕様書を書き切ってから実装に入る進め方です。Claude Codeと仕事をするときは、いきなりプロンプトから書き始めるのではなく、CLAUDE.mdとspec.mdで土台を用意するのが鍵です。Claude Codeは渡されたドキュメント通りに動くので、ドキュメントの質がそのまま出力の質になります。

---

## [P3] WHY — 月額数千円のSaaSが自前で作れる時代

名刺管理のSaaS、月額で使うWebサービスで代表的なのがSansanです。法人向け名刺管理で13年連続シェア1位、利用料は月額数千円から数万円。

アプリの機能の核は、データを登録・一覧・検索・編集・削除するCRUDと呼ばれる5つの操作です。この核だけなら、スペック駆動とClaude Codeの組み合わせで1時間で動くものが完成します。それを今日、体験します。

---

## [P4] CLAUDE.md — アプリ開発用CLAUDE.mdに書くこと

（画面を Claude Code デスクトップ版に切り替え、新しいプロジェクト用ディレクトリを作成）

アプリ開発用のCLAUDE.mdに書くことは6項目です。技術スタック、ディレクトリ構成、コーディング規約、データ構造、デザイン規約、そして「やってはいけないこと」。

（CLAUDE.mdファイルを画面に表示）

技術スタックはHTMLとJavaScript、それにTailwindのCDNだけ。ビルドツールもnpmも使いません。ファイルはindex.html、app.js、styles.cssの3枚で完結させます。小さく始めると、Claude Codeも迷いません。

続いてspec.mdです。画面構成はツーカラム、左に名刺の一覧、右に詳細とフォーム。CRUDの各操作フローと、localStorageに保存する方針も言語化します。ここまで決まれば、Claude Codeは迷わず実装に入れます。

---

## [P5] PROMPT

（スライドをP5に切り替え、プロンプトを読み上げ）

> CLAUDE.md と spec.md を読んで、
> 名刺管理アプリを実装してください。
>
> index.html / app.js / styles.css の
> 3ファイルで完結させてください。
>
> データはブラウザのlocalStorageに保存します。
> 実装が終わったらGitHubにプッシュしてください。

プロンプト本体はとても短いです。アプリの詳細は全部CLAUDE.mdとspec.mdに書いてあるので、ここでは「実装して」と頼むだけで済みます。

（Claude Codeにプロンプトを投入。ファイルが生成されていく）

実装が終わったら、ブラウザで開いて動かしてみます。

（実装したアプリを開き、カードを選択、新規登録、編集、削除を順に操作）

左に名刺の一覧、右に詳細。CRUDが全部動いています。データはブラウザのlocalStorageに入っているので、リロードしても消えません。

（Claude Codeに「GitHubにpushして」と依頼、もしくはターミナルでgit push）

GitHubへプッシュすると、Vercelで自動デプロイが走り、数十秒後にインターネットからアクセスできるURLが発行されます。

（Vercelの管理画面でデプロイ成功を確認し、発行URLを開く）

別の端末のブラウザでこのURLを開けば、同じアプリが動きます。

---

## [P6] クロージング — 自前のアプリが完成しました

月額のSaaSに払う代わりに、自前のアプリが手元に残りました。名刺を登録して検索して編集する、という日常の操作はひと通りできます。

SaaSとの違いがひとつあります。データがあなたのブラウザの中にしか残らないことです。別の端末からは見えませんし、他の人と共有もできません。ここを解決するのが、クラウドDBです。S6でSupabaseにデータを預けると、どの端末からでも同じデータが見えるようになります。
